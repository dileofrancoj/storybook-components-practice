# Get started

`lvlup-base-components` es una libreria que implementa componentes comunes dentro del curso de LVL UP

Los componentes definidos deben recibir solamente objetos de dominio para establecer una abstraccion
semántica basada en un lenguaje en común con los conceptos creados por el equipo de UX, obligando a
que los componentes sean totalmente declarativos. De esta forma logramos evitar tener que modficar
la implementación en las apps que consuman esta libreria si llega a cambiar el diseño visual.

## Install

#### Con yarn:

```bash
$ npx install-peerdeps @lvl-up-lib --yarn
$ yarn add @lvl-up-lib
```
